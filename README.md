# OmniGrowth — Pan-India B2B Agricultural Produce & Intelligence Network

OmniGrowth is a live interactive commodity intelligence map of India built for enterprise buyers, food processors, supermarkets, and agricultural bulk procurers.

## Features
- **Scroll-Driven Map Camera**: Powered by MapLibre GL JS & GSAP with dynamic pitch perspective.
- **Translucent Crop Coverage & Pinpoint Nodes**: Real-time visualization of crop category regions across India.
- **Auto-Flipping Glassmorphism Cards**: Displays top crop production metrics, soil NPK/pH profiles, and mandi price breakdowns.
- **Client-Safe Logistics Showcase**: Highlight 99.4% on-time B2B freight capabilities, cold chain reefer fleets, and APMC mandi pass automation.
- **Founder Profile**: Varshith Krishna (Founder & Managing Director).
- **SQLite + Node.js Native Backend**: Lightning-fast local database queries (<2ms).

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS v4, MapLibre GL JS, deck.gl, GSAP.
- **Backend**: Node.js v24, Express, TypeScript, `node:sqlite`.
